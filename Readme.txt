AI 仿製可愛通訊軟體貼圖(AI imitates cute communication software stickers)

資料來源: https://3c.ltn.com.tw/news/62802
https://creator.line.me/zh-hant/guideline/sticker/  ~ LINE 貼圖製作準則
https://creator.line.me/  ~ LINE 貼圖上架登入/註冊

步驟01. 要AI產生一張無版權的Q版貓頭鷹圖片 ~ 胖頭鷹家族日常

步驟02. 嘗試資料來源要AI產生老人問候圖 (對應提示詞) ~ 我想一次做9張Q版貼圖，以這張照片為原型，可愛風格。每張貼圖要搭配不同的表情、動作和文字，文字包含「早安、晚安、thank you、sorry、你好、加油、Byebye、OK、Good」，且每一個字詞對應的貼圖要符合情境。

創意發想:貓頭鷹和貓有87%相似，所以拿我最喜歡的白爛貓貼圖做為模仿對象 要AI 進行模仿抄襲

步驟03. 給AI一張模仿對象圖 和 無版權Q版貓頭鷹圖片 要他合成新圖片 (對應提示詞) ~ 使用第一張圖為創意來源 使用第二章圖為原型做出類似的Q版圖片
